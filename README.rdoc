A small class for validating Australian Business Numbers (ABN)

= Installation

  gem install yob-abn --source http://gems.github.com/

= Usage

  require 'abn'

  ABN.new("12042168743").valid?
  => true

  ABN.valid_abn?("12042168743")
  => true

  ABN.valid_abn?("12042168744")
  => false
